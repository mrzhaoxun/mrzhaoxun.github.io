Web服务器理解：uWSGI与Nginx配置

Web开发中是由Web服务器与Web框架两大块构成。

其中Web服务器即建立连接，转发请求与响应，对请求与响应的处理交由Web框架处理。

WSGI： Web服务器网关接口（Python Web Server Gateway Interface）

是Python程序或框架和Web服务器之间的一种接口、协议

uWSGI：

实现了WSGI协议的一个Web服务器。

实现过程：用户到服务器间是http协议，服务器到Python框架间是WSGI协议，这里就需要uWSGI服务器实现WSGI协议。同理反之，响应数据也由Python框架通过WSGI协议返回给uWSGI服务器，再通过http协议传递给用户。

  • uWSGI + Django 即完成了简单的Web开发（仅适用访问量较少的个人网站）。

pip install uwsgi

常用指令：

uwsgi –ini uwsgi.ini #按配置文件初始化

uwsgi –stop uwsgi.pid #按pid信息关闭响应uwsgi服务器进程

vim uwsgi.log #查看日志
