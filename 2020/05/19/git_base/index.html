Git基础操作

Git 安装及基础配置

sudo apt-get install git

#mysql进程出错，在其配置文件中添加

#[mysql]
#protocol=tcp

  1. 配置git全局环境

git config –global user.name “用户名”

git config –global user.email “邮箱地址”

​ 查看设置

git config –list

  1. 生成ssh密钥（配置到GitHub中）

ssh-keygen -C ‘you email 936330052@qq.com‘ -t rsa

  1. 创建文件夹，初始化仓库

git init

​ 测试连接

ssh -T git@github.com

github克隆

git clone git@github.com:mrzhaoxun/e-commerce-django.git

WorkSpace → Stage → Local Repo → Romate Repo

git add / git add . / #从WorkSpace保存到Stage,add后的文件才会被git跟踪

git status #查看WorkSpace的状态

git diff #显示WorkSpace和Stage中的状态差异

git mv #文件改名

git rm #删除文件

git rm –cached #从Stage中移除，停止跟踪文件但不删除

git commit -m “message” #从Stage提交更新到Local Repo

git push -u origin master #上传代码并快速合并

git pull origin master #下载代码并快速合并

清理缓存区

git rm –cached readme.txt #指定文件

rm .git/index #清空
