搭建个人hexo博客的第一步

hexo搭建个人博客，初步部署到GitHub

为便于操作切换root

sudo su

第一步：安装hexo所需环境 node 、npm 、cnpm

1. 下载node.js（包含node, npm）

查看版本信息，检查安装成功

not found重新设置软连接：

sudo ln -s /usr/local/node-v12.16.3-linux-x64/bin/node /usr/local/bin/

sudo ln -s /usr/local/node-v12.16.3-linux-x64/bin/npm /usr/local/bin/

2. 安装cnpm

npm install -g cnpm –registry=https://registry.npm.taobao.org

#软连接出错解决：

sudo ln -s /usr/local/node-v12.16.3-linux-x64/bin/cnpm /usr/local/bin/

第二步：Git（默认已经安装）

详细可查阅[Peter的git指南]

第三部： hexo

创建一个blog文件夹，方便管理

1. 下载hexo 、初始化

cnpm install -g hexo-cli

hexo init #blog文件夹下

2. hexo 操作：

hexo s 启动 n 创建 g 生成 d 部署 clean 清理

3. 部署github
  1. 创建仓库 命名：<github用户名>.github.io
  2. blog文件夹下 安装git 部署插件

cnpm install –save hexo-deployer-git

  1. 设置_config.yml

#添加仓库地址：

type: git

repo: https://github.com/mrzhaoxun/mrzhaoxun.github.io.git

branch: master

  1. 部署 hexo d

  2. 更换主题

    #克隆他人主题

    git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

配置_config.yml 主题

theme: yilia
